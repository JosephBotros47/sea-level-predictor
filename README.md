## Project Overview
This project is part of the freeCodeCamp Data Analysis with Python Certification. It analyzes a dataset of the global average sea level change since 1880 and uses the data to predict the sea level change through the year 2050 using linear regression.

## Tasks Completed
- Imported historical sea level data using Pandas.
- Created a scatter plot of the data.
- Calculated the line of best fit based on all historical data and extended it to the year 2050.
- Calculated a second line of best fit based on recent data (from year 2000 onwards) to predict future trends if the rate of rise continues at its current accelerated pace.
